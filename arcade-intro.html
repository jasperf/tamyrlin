Meet Arcade

FUNDAMENTALS

What is Arcade?

Arcade is a modern Python library for creating 2D video games. It's designed to be easy to learn while still being powerful enough for complex games like platformers, RPGs, and puzzle games.

Why Arcade over Pygame?

  • Simpler API - Cleaner, more intuitive functions
  • Built-in camera - Easy scrolling for large worlds
  • Physics engine - Simple collision and movement
  • Modern - Actively maintained with good documentation

Installation

Open Terminal on your Mac and run:

pip3 install arcade

This installs the Arcade library. You only need to do this once.

Verify it worked: Run python3 -c "import arcade; print('Arcade is ready!')"

The Game Loop

Every Arcade game follows this pattern:

import arcade class MyGame(arcade.Window): def __init__(self): super().__init__(800, 600, "My Game") arcade.set_background_color(arcade.csscolor.DARK_BLUE) def on_draw(self): # Draw everything here self.clear() arcade.draw_circle_filled(100, 100, 50, arcade.csscolor.RED) game = MyGame() arcade.run()

Key concepts:

  • Window(width, height, title) - Creates the game window
  • on_draw() - Called every frame to draw graphics
  • arcade.run() - Starts the game loop

Coordinate System

Arcade uses a standard Cartesian coordinate system:

  • Origin (0, 0) is at the bottom-left corner
  • X-axis goes right (positive) and left (negative)
  • Y-axis goes up (positive) and down (negative)

HANDS-ON CODING

Your First Arcade Program

Now it's your turn! Create a simple Python file that opens a game window.

TASK

Create game.py

Create a new file called game.py with the following code:

import arcade class Game(arcade.Window): def __init__(self): super().__init__(800, 600, "My Platformer") arcade.set_background_color(arcade.csscolor.DARK_BLUE) def on_draw(self): self.clear() arcade.draw_text("Hello, Arcade!", 400, 300, arcade.csscolor.WHITE, 36) window = Game() arcade.run()
  • Create the file: Open Terminal and run nano game.py (or use VS Code)
  • Paste the code: Copy the code above into your file
  • Save: Press Ctrl+O, then Enter to save in nano, or just save in your editor
  • Run it: In Terminal, run python3 game.py

    • Troubleshooting

    • ModuleNotFoundError: You forgot to run pip3 install arcade
    • Command not found: Make sure you're using python3 not python
    • Window closes immediately: You might have forgotten arcade.run()
    • Syntax error: Check for typos in the code
